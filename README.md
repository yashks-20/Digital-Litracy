## Project Overview

This project is part of the Digital Literacy course. The goal of this project is to understand how to use digital tools effectively, maintain a professional online presence, and stay safe in the digital world.

As a Digital Ambassador, this project covers different aspects of digital literacy including awareness, online platforms, communication, and cyber safety.

---

## Task Summary

### Task 1 – Digital Literacy Awareness Infographic

Created an infographic using Canva covering topics like digital literacy, safe internet practices, and useful digital tools. This helped in understanding how to present information clearly in a visual format.

---

### Task 2 – Student Digital Portfolio

Created and updated profiles on platforms like GitHub and LinkedIn. Learned how to maintain a professional online presence and how these platforms can be useful for future opportunities.

---

### Task 3 – Coding & Collaboration Platforms

Completed a beginner-level coding challenge on a platform like HackerRank. Also created a Google Form for a digital literacy quiz and understood how collaboration tools can be used effectively.

---

### Task 4 – Email Etiquette & Communication

Wrote professional emails and created a checklist for responsible social media usage. This task helped in understanding proper communication and online behavior.

---

### Task 5 – Cybercrime Awareness

Studied a cybercrime case and created a prevention checklist. Learned about online threats and how to stay safe while using digital platforms.
